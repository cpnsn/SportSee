# SportSee Front-end

## Prerequisites

[Npm](https://www.npmjs.com/)

## Launching the project

Fork the repository.\
Clone it on your computer.\
The `npm install` command will allow you to install the dependencies.\
The `npm start` command will allow you to run the app.
